# Хамса — Статический сайт

Простой HTML/CSS/JS сайт, который можно редактировать прямо на GitHub без сборки!

## 🌐 Демо

https://santo4ka.github.io/xamsa2/

## 📁 Структура файлов

```
xamsa2/
├── index.html      ← Основной файл (тексты, структура)
├── style.css       ← Стили (цвета, размеры, отступы)
├── script.js       ← Интерактивность (меню, табы, форма)
├── favicon.svg     ← Иконка сайта
└── images/         ← Папка с фото
    ├── hamsa-hero.jpg
    ├── product-light.jpg
    ├── product-spicy.jpg
    ├── gallery-1.jpg ... gallery-6.jpg
    └── ...
```

## ✏️ Как редактировать на GitHub

### 1. Поменять цены

Открой `index.html` и найди блоки с ценами:

**Цены за килограмм (в карточках продукции):**
```html
<!-- Строка ~134 -->
<div class="product-price">
  <span>Цена:</span>
  <strong>от 180 ₴/кг</strong>  ← Поменяй здесь
</div>
```

**Цены по фасовке:**
```html
<!-- Строка ~192 -->
<div class="pricing-price">90 <span>₴</span></div>  ← 0.5 кг малосольная

<!-- Строка ~205 -->
<div class="pricing-price">170 <span>₴</span></div>  ← 1 кг малосольная

<!-- Строка ~218 -->
<div class="pricing-price">400 <span>₴</span></div>  ← 2.5 кг малосольная
```

Также поменяй в выпадающем списке формы (строки ~325-330):
```html
<option value="light-0.5">Малосольная — 0.5 кг (90 ₴)</option>
```

### 2. Убрать блок

Просто удали HTML-код блока. Например, чтобы убрать **Галерею**:

1. Найди в `index.html`:
```html
<!-- Галерея -->
<section class="gallery" id="gallery">
  ...
</section>
```

2. Удали ВЕСЬ блок от `<section class="gallery"...>` до `</section>`

3. Не забудь убрать ссылку из меню:
```html
<li><a href="#gallery">Галерея</a></li>  ← Удали эту строку
```

### 3. Поменять текст

Просто найди текст в `index.html` и отредактируй:

```html
<h1>Свежая малосольная<br><span class="accent">и пряная хамса</span></h1>
<!-- Поменяй на: -->
<h1>Свежая рыба<br><span class="accent">с доставкой</span></h1>
```

### 4. Поменять цвета

Открой `style.css`, в начале файла есть переменные:

```css
:root {
  --color-primary: #0A4B7A;    ← Основной синий
  --color-accent: #00A8C6;     ← Бирюзовый
  --color-orange: #F5A623;     ← Оранжевый (кнопки)
  --color-navy: #0A2540;       ← Темно-синий (текст)
}
```

### 5. Заменить изображения

1. Загрузи новое фото в папку `images/` на GitHub
2. В `index.html` поменяй путь:
```html
<img src="./images/новое-фото.jpg" alt="Описание">
```

## 🚀 Как загрузить на GitHub Pages

1. Создай репозиторий `xamsa2`
2. Загрузи ВСЕ файлы (index.html, style.css, script.js, favicon.svg, папку images/)
3. В настройках репозитория: **Settings** → **Pages** → **Source** → выбери ветку `main`
4. Готово! Сайт будет по адресу: `https://santo4ka.github.io/xamsa2/`

## 📱 Быстрое редактирование на GitHub

1. Открой файл в репозитории
2. Нажми карандаш ✏️ (Edit this file)
3. Внеси изменения
4. Нажми **Commit changes...**
5. Сайт обновится автоматически через 1-2 минуты!

## ❓ Вопросы?

Если что-то не работает — проверь:
- Все файлы загружены в корень репозитория
- Пути к изображениям начинаются с `./images/`
- Нет опечаток в HTML-тегах
